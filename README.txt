Hangman the Game ltd.
Spelet Hangman körs genom terminal.
Detta är ett spel som handlar om att gissa rätt ord utifrån ett hemligt ord. 
Du kan gissa fel 8 gånger, efter det förlorar du spelet. 
För att vinna måste du gissa alla bokstäver som finns i det hemliga ordet. 
Lycka till!


Att bygga och köra.

Om du har maven på datorn kan du använda dig av kommandot:
mvn clean package

annars kan du köra våra jar-fil med kommandot:
java -jar ./target/Hangman-1.0-SNAPSHOT.jar
